## 📋 Prerequisites

On x86_64 architecture :
- Vagrant `2.3.7`
- virtualbox `>= 6.1`

On Apple Silicon / ARM architecture :
- Vagrant `2.3.7`
- VMWare Fusion `13.0.2`

## 🔧 Install

```sh
$ git clone https://github.com/mathias-mrsn/inception_of_things.git
```
